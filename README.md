# ESP32-CAM RC Car 🚗📷

A WiFi controlled RC car with live camera streaming using ESP32-CAM.

## Features
- Live video streaming
- Wireless control
- Mobile friendly interface
- ESP32 based system

## Hardware
- ESP32-CAM
- L298N Motor Driver
- DC Motors
- Battery
